## Navigation Features

### Header
- **Reset View**: Returns canvas to original position and scale
- **Resume**: View detailed resume page
- **Contact**: Access contact information and form
- **Terminal**: Click for quick command-line style info
- **Profile Avatar**: Click to open dropdown with social links

### Sidebar Controls
- **Zoom In**: Increase canvas scale up to 2.5x
- **Zoom Out**: Decrease canvas scale down to 0.5x
- **Share**: Share portfolio link via Web Share API or clipboard
